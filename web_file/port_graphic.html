3D Graphic-.

hands

Date : 2015.10.28
Graphic Tool : Autodesk MAYA, Adobe Photoshop CS3 Extended
Description : 2학년이 되어 수강하게 된 '컴퓨터그래픽스' 과목의 수업 방식은 특별했습니다.
교수님께서는MAYA라는 프로그램을 설치하게 하신 후 독학으로 프로그램을 공부하는 것을 과제로 내주셨습니다.
그리고 이를 평가하기 위해 자신의 손을 만들어 제출하는 것을 프로젝트로 하여 성적을 주신다고 했습니다.
이 작업이 의미를 가지는 이유는, 제가 처음으로 MAYA라는 툴을 이용하여 만들어낸 작품이라는 의의 때문입니다.
